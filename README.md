# Calculator App — QA Automation Suite

## 📖 Overview
Automated test suite for a **Calculator Web Application** using Selenium WebDriver, Java, and TestNG. Covers arithmetic operations, edge cases, and UI validation with comprehensive positive and negative test scenarios.

---

## 🚀 Key Testing Highlights
- **Arithmetic Operations:** Validated addition, subtraction, multiplication, and division
- **Edge Case Testing:** Handled division by zero, large numbers, and decimal precision
- **Operator Chaining:** Tested multiple sequential operations
- **Clear/Reset Validation:** Verified CE, C, and backspace button behaviors
- **Negative Testing:** Invalid input handling and boundary condition validation
- **Data-Driven:** Parameterized test cases using TestNG DataProvider
